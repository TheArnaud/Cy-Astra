Accès au site :

url : cy-astra.fr ou https://cy-astra.fr/


Manipulation pré-requise pour utiliser le site localement:

Avant toute chose veuillez ajouter à votre base de données (SQL) une base de donnée "projet" et les tables contenus 
dans le fichier "database".
Celle-ci permettent de voir toutes les images , objets de boutiques , événements et photo de la galerie
Il est donc très important de le faire.

Structure du dossier :

Le dossier "ProjetInfo" contient directement les différents fichiers .php qui forment les différents onglets du site 
(accueil.php , boutique.php , events.php etc...) ainsi que leurs .css respectifs
Ce dossier contient aussi deux sous-dossiers : "database" , qui contient les différentes tables de la base de données comme expliqué précédemment
					       "images" , qui contient toutes les images utilisées sur le site et 2 sous-dossiers pour la boutique et galerie


Admin :

Pour se connecter en tant qu'administrateur et avoir les priviléges, cliquez sur connexion -> admin --> mots de passe : AdminMain
